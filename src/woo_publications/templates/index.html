De Publicatiebank en bijhorende API bevatten en ontsluiten overheidsdocumenten in het kader van de Wet Open Overheid (WOO). Deze registratiecomponent maakt het mogelijk voor:

  • publicatiecomponenten om publicaties en documenten te beheren
  • burgerportalen om publicaties weer te geven en indexeren in de landelijke index

Dit component bevat slechts een minimale beheeromgeving. Voor productie-doeleinden dien je een publicatiecomponent te koppelen.

The publications database and accompanying API contain and expose government documents intended for publication in the context of the "Open Government Law", for which the Dutch abbreviation is "WOO". This registration component enables:

  • publication components to manage publications and documents
  • citizen portals to display publications and have them indexed in the national index

This component contains only a minimal administrative interface. For production purposes you need to connect a publication component.
