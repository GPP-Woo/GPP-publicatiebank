Installation
============

You can install WOO Publications in several ways, depending on your intended purpose
and expertise.

1. Deploy using :ref:`Helm charts (TODO) <installation_helm>`
2. Run with :ref:`Docker Compose <installation_docker_compose>` on your computer for
   private testing purposes
3. Run from :ref:`Python code <developers_installation>` on your computer for
   development purposes

After deploying the application, review the :ref:`installation_requirements` and  make
sure to finish the installation with the necessary :ref:`configuration <configuration_index>`.
